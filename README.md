## Instructions
   
   To compile AND link AND run: 

      ```
      $ make run
      ```

   To compile AND link: 

      ```
      $ make
      ```

   To compile only: 

      ```
      $ make p1.o
      ```

   To clean dir of .o, executable, and tilda (~) files: 

      ```
      $ make clean
      ```
